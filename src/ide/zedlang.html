The Zed Programming Language

What is a Program?


PROG Program_Name
  OUT["Hello, World"]:
ENDPROG

Instruction Sequence

Assignment, Output, WHEN, REPEAT, FOR, IF, SWITCH

Assignment


= A1 "Hello, World!":
= B1 42:
= C1 IN["What's your name? "]:

Output


OUT["Hello, World"]:

WHEN loop


= I1 0:
WHEN < I1 9
  DO
    = I1 1+:
    OUT[I1]:
  ENDDO
ENDWHEN

REPEAT loop


  = I1 0:
  REPEAT
    DO
      = I1 1+:
      OUT[I1]:
    ENDDO
    UNTIL > I1 9
  ENDWHEN

FOR loop


FOR I1 FROM 1 TO 10 BY 1
  DO
    OUT[I1]:
  ENDDO
ENDFOR

IF statement


= X1 1:
IF == X1 1
  DO
    OUT["ONE"]:
  ENDDO
OTHERWISE IF == X1 2
  DO
    OUT["TWO"]:
  ENDDO
OTHERWISE
  DO
    OUT["ANYTHING"]:
  ENDDO
ENDIF

SWITCH statement


= X1 "apple":
SWITCH X1
  WHEN "apple"
    DO
      OUT["Le Pomme"]:
    ENDDO
  WHEN "orange"
    DO
      OUT["Le Citrus"]:
    ENDDO
ENDSWITCH

Comparative Conditions


< 1 2:   # true
> 1 2:   # false
<= 2 2:  # true
>= 3 2:  # true
== 1 1:  # true

Logical Conditions


&& == 1 1 == 0 1: # false
|| == 1 1 == 0 1: # true
! && == 1 1 == 0 1: # true
! || == 1 1 == 0 1: # false
